# 📻 点划之间 - UI/UX 设计与体验解析

在 **点划之间** 的设计中，我们构建了一套名为「OLED 深邃黑 × 鎏金光影」的沉浸式视觉系统。通过细腻的微交互、物理级触觉反馈与拟真音频合成，让每一次敲击都成为跨越时空的对话。

## 一、 色彩美学：深空与鎏金的交响

我们摒弃了传统的明亮模式，采用了专为沉浸感与专注力打造的 **OLED 暗黑主题 (Dark Mode)**，并辅以极具复古奢华感的**金属金系 (Metallic Gold)** 作为核心品牌色。

- **🌌 极光黑背景 (Deep Space Background)**
  使用带有微弱径向渐变的深邃黑（`#0b0b0f` 至 `#08080c`）作为基底。不仅在 OLED 屏幕上极致省电，更能像夜空般衬托出主要交互元素的层级关系，减少视觉疲劳。
- **🏆 鎏金主色调 (Vintage Gold Accent)**
  从温润白金（`#faf3df`）到深古铜色（`#4e3b17`），构成了丰富的金属渐变光谱。核心交互（如 Tab 栏、主按钮）大量使用高光鎏金渐变，完美模拟出黄铜电键在灯光下的细腻光泽。
- **📝 文本与语义 (Text & Semantics)**
  主文本使用柔和的米白色（`#f2efe8`），避免纯白刺眼；辅助信息采用静谧灰（`#9a9a9f`）。状态反馈则使用明快的翡翠绿（Success）与警示的珊瑚红（Error）。

## 二、 字体排版：古典与现代的碰撞

字体的选择直接决定了产品的气质。我们精心挑选了三种不同性格的字体，构建出层次分明的阅读体验：

- **优雅衬线 (Display) - `Cormorant`**
  用于展示大号字母与标题。其修长的笔画和古典的韵味，宛如泛黄电报纸上的印刷体，瞬间拉满复古学术氛围。
- **清晰无衬线 (Body) - `Inter`**
  用于界面中的正文、按钮与提示。极致的现代感与极高的可读性，确保了 UI 的干净利落。
- **等宽极客 (Mono) - `JetBrains Mono`**
  用于分数、连击数等数据展示。等宽特性不仅排版整齐，更赋予了产品一丝破译密码的极客神秘感。

## 三、 视觉特效：光影流转的微交互

静态的 UI 是骨架，动态的光影则是灵魂。**点划之间** 在细节处藏满了令人愉悦的视觉魔法：

- **🪟 轻量级毛玻璃 (Light Glassmorphism)**
  导航栏与悬浮卡片采用高斯模糊处理（`backdrop-filter: blur(18px) saturate(140%)`），在深色背景上营造出晶莹剔透的磨纱质感，轻盈而不喧宾夺主。
- **🌟 呼吸光环与动态光晕 (Dynamic Glow & Breathe)**
  待机状态下，主电键外围环绕着一圈微弱的「呼吸光环」，仿佛电报机在静静等待指令。当用户按下电键时，会根据**按压时长与强度**，动态渲染出金色光晕的扩散与提亮，让敲击能量完全可视化。
- **💧 触控涟漪 (Tactile Ripples)**
  每次点击按钮，都会以触控点为中心激荡出柔和的径向渐变涟漪，提供即时且符合直觉的视觉确认。
- **🔥 连击与成就动效 (Combo Animations)**
  答对时，带有金色流光的积分会轻盈上浮；达成连击（Combo）时，专属的成就横幅会伴随金色火花闪耀登场；答错时，则会有真实的屏幕物理震动与红色警示闪烁。

## 四、 沉浸感官：听觉与触觉的通感

真正的硬件级体验，离不开声音与震动的完美配合：

- **📻 拟真 CW 电报音合成 (Authentic Audio Synthesis)**
  摒弃了干瘪的普通蜂鸣声，我们使用 Web Audio API 构建了复杂的合成器：**660Hz 基频 + 泛音/次谐波 + 5Hz 轻微颤音 (LFO) + 低通滤波柔化**。配合 22ms 的软起音彻底杜绝了刺耳的爆音，听感无限逼近真实的 HAM 无线电报机，温暖且醇厚。
- **⚡ 蓄力发声 (Expressive Keying)**
  长按发报时，声音并非一成不变。随着按压时间的延长，**音频的滤波器会逐渐打开（音色变亮），基频微升（660Hz -> 700Hz）**，配合视觉上的光晕扩散，完美模拟出一种「能量蓄力」的物理阻尼感。
- **📳 物理级触觉反馈 (Haptic Engine)**
  深度调用设备的震动马达：短按（Dot）对应轻巧的“嗒”，长按（Dash）对应沉稳的“哒”。答对时的欢快震动与答错时的沉重闷震，让每一次交互都拥有了真实的重量。
- **🎵 奖励音效 (Rewarding Soundscapes)**
  答对时的三音上扬和弦清脆悦耳，连击里程碑的短促金铃铛声不断刺激多巴胺分泌，让枯燥的摩斯密码学习变成了一场停不下来的游戏。
