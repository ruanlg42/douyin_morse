# 默认曲摩斯动机融合优化设计

## 背景
当前互动空间默认曲（LOVE/HOME/STAR/DREAM）的前几下摩斯敲击听感偏割裂，像后贴音效而非歌曲内部动机。目标是在保留摩斯可辨识度的同时，让它更自然地融入编曲。

## 方向
采用“温柔融合版”：摩斯仍可听出点划，但需要像旋律/节奏动机，避免干硬、突兀、过近的 attack。

## 改造点
1. 首轮 hook 延后进入：保留 0.8-1.2 秒 pre-roll，让底乐或氛围先建立。
2. 软化音色：为 hook 音符增加更柔和的 attack/release，并降低首轮亮度和干声感。
3. 分段音量重平衡：intro 段下调，verse 维持低存在感，chorus 才轻微抬升。
4. 节拍对齐：首轮与后续循环按小节/半拍网格吸附，避免从 0ms 生硬切入。
5. 产物更新：重生成 4 首默认歌曲，更新 manifest、morse_app.zip，并通过 h5-validator。

## 非目标
不引入实时后端到互动空间包；不降低游戏与关卡码功能；不追求完全隐藏摩斯。
