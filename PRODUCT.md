# 产品功能总览

面向用户的 **摩斯码 × 音乐 × 解压** 一体化 Web 应用（内部常称「声印」体系）。  
同一套后端提供 API 与静态资源，前端为 **单页三 Tab**，学习、创作与轻量小游戏并列，无需分别安装多个应用。

---

## 产品定位

- **学习**：用视频与发报手感建立「字母 ↔ 摩斯」直觉，带测验与反馈。  
- **创作（声印）**：输入英文词 → 生成带摩斯前奏意象的短音乐，内置碟中谍示例。  
- **解压**：基于电键点划的轻交互（当前为「摩斯揍字机」等玩法迭代中），强调趣味与发泄，不强调考核。

技术气质：**深色 + 金玻璃拟态**、触觉反馈、Web Audio 电键音、移动端友好布局。

## 三大 Tab 功能说明

### 1. Learn（学习）

| 能力 | 说明 |
|------|------|
| **字母视频** | 每个字母对应 `frontend/public/letter/*.mp4`，画中展示该字母的摩斯点划，辅助记忆。 |
| **发报电键** | 大圆形电键：短触为「点」、长按为「划」；带触觉、蓄力光效与 CW 风格音色。 |
| **声音** | 视频默认尝试有声播放；浏览器策略限制时可能需用户点喇叭或播放解锁。 |
| **播放模式** | 字母顺序播放 / 单字母循环。 |
| **学习 / 测试** | 学习模式跟视频与参考；测试模式给大写字母，隐藏答案形状，输入摩斯判对错、连击与星级等游戏化反馈。 |

### 2. Music（声印 · 音乐）

| 能力 | 说明 |
|------|------|
| **示例曲** | 一键加载「碟中谍」主题向示例：内置时间轴、前奏摩斯动画、示例音频（`backend/morse_api/assets/mission.mp3`），**不消耗 AI**。 |
| **自定义生成** | 输入 1～10 个英文字母（具体规则以后端校验为准），选风格、可选人声，请求 `/api/generate`。 |
| **摩斯前奏** | 服务端将词转为摩斯，合成鼓点/节奏轨，再与 AI 生成的旋律段落混音。 |
| **播放器** | 封面图：示例用电影海报风资源，用户生成用品牌插画；进度条可拖拽；主控为播放/暂停。 |
| **词组演示** | 示例支持 `display_phrase` 等时间轴高亮（如 Mission: Impossible 式逐字动效）。 |

### 3. 解压（原 Play）

| 能力 | 说明 |
|------|------|
| **电键输入** | 与学习页一致的点划逻辑与音色（沙盒内可独立迭代玩法）。 |
| **当前玩法方向** | 停拍自动识别摩斯字母 → 角色化动画（如挨揍、变形）→ 计数与轻量文案；具体以仓库内最新 `PlayScreen` 为准，可持续换皮为其他解压机制。 |

## 后端能力（FastAPI）

| 接口 / 资源 | 作用 |
|-------------|------|
| `GET /api/health` | 健康检查。 |
| `GET /api/styles` | 音乐风格列表。 |
| `GET /api/demo` | 返回碟中谍示例 JSON（含 `demo: true`、时间轴、`audio_url` 等）。 |
| `POST /api/generate` |  body：`word`、`style`、`with_vocals` → 生成混音后的 MP3，并返回时间轴与元数据。 |
| `GET /media/*` | 用户生成音频等输出文件。 |
| `GET /assets/*` | 示例静态资源（如 `mission.mp3`）。 |

**AI / 模型（MiniMax，同一 API Key）**

- **音乐生成**：曲风提示与人声歌词等链路（见 `minimax_client.py`）。  
- **文本大模型**：已实现 `text_chat_completion_v2`（如 `M2-her`），当前主要用于**音乐提示词与歌词生成**；若要做「小游戏对话」等，需**新增独立 API 路由**再对接前端。

---

## 前端能力摘要

- **框架**：React 18 + Vite 6 + Tailwind 3。  
- **开发代理**：`/api`、`/media`、`/assets` → 后端（见 `vite.config.js`）。  
- **生产构建**：`npm run build`，资源目录 `build.assetsDir = 'bundle'`，与后端 `/assets` 路径区分；可选环境变量 `MORSE_SPA_DIST` 由 FastAPI 挂载整站 SPA（见 `main.py`）。  
- **无障碍**：主要按钮带 `aria-label`、部分 `prefers-reduced-motion` 处理。
